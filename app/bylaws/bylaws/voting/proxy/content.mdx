## Proxy Voting

Procedures for proxy voting in organization elections.

### Authorization Process

- Member must submit written request
- Authorization valid for single election only
- Proxy must be in good standing

### Limitations

- One proxy per member
- No proxy chaining allowed
- Board can revoke proxy rights
